# Hypemove Landing (Vite + React + Tailwind)

## Setup
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

`npm run build` ora genera prima il bundle Vite e poi prerendera la homepage dentro `dist/index.html`, cosi la landing restituisce HTML gia popolato anche senza eseguire JavaScript.
